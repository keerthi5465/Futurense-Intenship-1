    <h1>Database Commands Overview</h1>
    
    <h2>Data Definition Language (DDL)</h2>
    <p>DDL commands are used to define and manage the structure of the database and its objects:</p>
    <ul>
        <li><strong>CREATE:</strong> Used to create new objects like tables, indexes, views, etc.</li>
        <li><strong>ALTER:</strong> Used to modify the structure of existing objects.</li>
        <li><strong>DROP:</strong> Used to delete or remove existing objects.</li>
        <li><strong>TRUNCATE:</strong> Used to remove all records from a table while preserving its structure.</li>
    </ul>
    
    <h2>Data Manipulation Language (DML)</h2>
    <p>DML commands are used to manage the data within the database:</p>
    <ul>
        <li><strong>SELECT:</strong> Used to retrieve data from one or more tables.</li>
        <li><strong>INSERT:</strong> Used to add new records into a table.</li>
        <li><strong>UPDATE:</strong> Used to modify existing records in a table.</li>
        <li><strong>DELETE:</strong> Used to remove records from a table.</li>
    </ul>
    
    <h2>Data Query Language (DQL)</h2>
    <p>DQL is used to query the database for information:</p>
    <ul>
        <li><strong>SELECT:</strong> The primary command used to retrieve data from databases.</li>
    </ul>
    
    <p class="highlight">Understanding these commands is essential for effectively managing databases and working with SQL-based systems.</p>
